# Code of Conduct

The TerraScope project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

In short:

- Be kind, respectful, and patient with everyone.
- Assume good faith. Disagree with ideas, not people.
- No harassment, no discrimination, no personal attacks.

The full text is at the link above. Violations can be reported to **conduct@terrascope.app**. All reports are confidential.
