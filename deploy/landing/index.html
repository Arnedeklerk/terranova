Classify Earth, gracefully.

A modern QGIS plugin for semi-automatic classification, foundation-model inference, and time-series change detection. STAC-first, COG everywhere, free and open source.

Install in QGIS Source on GitHub

What's in the box

  • STAC search Planetary Computer, Earth Search, CDSE. Lazy xarray stacks; download only what you need.
  • Classification Random Forest, LightGBM, XGBoost, MLP. One-click accuracy PDF.
  • Foundation models Prithvi-EO-2.0, Clay v1.5, TerraMind via TerraTorch. Export to ONNX.
  • SAM 3 Text and click prompts via segment-geospatial.
  • Time-series BFAST, LandTrendr, CCDC per-pixel. MP4 export.
  • Modern UI Command palette, wizards, dark mode, embedded React via QtWebEngine.

Privacy

Telemetry is off by default. If you opt in, we send six fields and nothing else — event name, plugin version, QGIS version, OS, an opaque installation id, and a timestamp. No imagery, no AOIs, no file paths, no credentials, no IP addresses. Read the policy →
