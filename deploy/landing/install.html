TerraScope

Install

QGIS 3.40 LTR or any 4.x.

From the QGIS Plugin Manager

  1. Open QGIS.
  2. Plugins → Manage and Install Plugins → All.
  3. Search for TerraScope.
  4. Click Install Plugin.

Optional extras

Foundation-model fine-tuning and SAM prompts need extra Python packages. Install via the OSGeo4W shell: 

python -m pip install \
  pystac-client planetary-computer odc-stac \
  scikit-learn reportlab matplotlib \
  segment-geospatial \
  torch terratorch lightning \
  zarr dask imageio-ffmpeg

From source

git clone https://github.com/terrascope-rs/terrascope
cd terrascope
make ui-build
make deploy

See the development guide for the full setup.
