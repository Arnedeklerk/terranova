about=TerraScope is a STAC-first, COG/xarray-powered QGIS plugin for Earth observation. It provides classical and deep-learning classification (Prithvi-EO-2.0, Clay, SAM 3 via TerraTorch), Sentinel/Landsat search and download from Planetary Computer / Earth Search / Copernicus Data Space, time-series change detection (BFAST, LandTrendr, CCDC), and a modern Fluent-style UI with optional embedded React panels via QtWebEngine. Free and open source under GPL-3.
